# Cleverlocale

Cleverlocale — a [Next.js](https://nextjs.org) app (bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app)) with Prisma.

## Getting started

Install dependencies, configure `.env`, then run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Edit `src/app` routes; the dev server hot-reloads.

## Database

```bash
npm run db:migrate   # or db:push for quick prototyping
npm run db:seed
npm run db:studio
```

## Build

```bash
npm run build
npm start
```
